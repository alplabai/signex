> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Turn four footprint-editor active-bar stubs (Move-by-X,Y, custom filter presets, bounding-box text frame, 3D body mint) into real, tested behaviour.

**Architecture:** Each item wires an existing `FootprintActiveBarStub` dropdown entry to a real `PrimitiveEditorMsg` handled in `dispatch/library.rs` via `editor.with_parts(|state, primitive| …)` with `push_history` / `dirty` / `canvas_cache.clear()`. Two items add a small schema field (`signex-library`); one adds a footprint-native preset type + prefs I/O. All geometry mutations mirror the proven `FootprintActiveBarNudgeSelection` pattern.

**Tech Stack:** Rust, Iced 0.14, `signex-app` (editor + dispatch), `signex-library` (footprint schema), `signex-bake` (unchanged — already consumes `Body3D`).

}

```rust
/// Translate the current pad selection by (dx, dy) mm: history snapshot,
/// tested `nudge_pads`, sketch mirror, primitive re-sync. No-op on empty
/// selection. Shared by the one-step nudge and the Move-By modal.
fn footprint_nudge_selection(editor: &mut FootprintEditor, dx: f64, dy: f64) {
    let mut indices: Vec<usize> = Vec::new();
    if let Some(p) = editor.state.selected_pad {
        indices.push(p);
    }
    indices.extend(editor.state.selected_pads_extra.iter().copied());
    indices.sort_unstable();
    indices.dedup();
    indices.retain(|&i| i < editor.state.pads.len());
    if indices.is_empty() {
        return;
    }
    editor.push_history();
    editor.with_parts(|state, primitive| {
        let moved = state.nudge_pads(&indices, dx, dy);
        let snapshots: Vec<crate::library::editor::footprint::state::EditorPad> =
            moved.iter().filter_map(|&i| state.pads.get(i).cloned()).collect();
        for snapshot in &snapshots {
            pad_to_sketch::mirror_move_pad_in_sketch(snapshot, primitive);
        }
        CanvasState::sync_pads_to_primitive(state, primitive);
    });
    editor.canvas_cache.clear();
    editor.dirty = true;
}
```
